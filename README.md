Hướng dẫn cài đặt và khởi chạy

Yêu cầu chuẩn bị

1.Đảm bảo máy tính của bạn đã có:

-Trình duyệt: Google Chrome, Microsoft Edge, hoặc Safari.

-Trình soạn thảo: Visual Studio Code (VS Code) được khuyến nghị.

-Tiện ích mở rộng: Live Server (trên VS Code) để xem thay đổi trực tiếp khi sửa code.

2.Tải mã nguồn:

-Cách 1: sử dụng git (git clone https://github.com/khiem0504/BTN-web.git)

-Cách 2: Tải thủ công: Nhấn vào nút Code màu xanh trên trang GitHub này và chọn Download ZIP, sau đó giải nén tệp tin.

3.Cách chạy code

-Mở thư mục dự án bằng Visual Studio Code.

-Tìm tệp tin có tên là home.html trong danh sách tệp tin.

-Kích chuột phải vào tệp home.html và chọn Open with Live Server.

-Trình duyệt sẽ tự động mở trang chủ của website.
